
# Simple Case & Roulette Telegram Bot (aiogram)

## Описание
Простой бот на Python + aiogram с:
- Балансом (⭐ Stars) и внутренним TON балансом
- Магазином кейсов
- Открытием кейсов (добавляет предметы в инвентарь)
- Продажа предметов за TON
- Рулетка с текстовой анимацией
- Реферальная система при /start referral_user_id
- Простая админ панель (ADMIN_ID в env)

## Быстрый старт
1. Скопируйте проект
2. Установите зависимости: `pip install -r requirements.txt`
3. Задайте переменные окружения:
   - `TG_BOT_TOKEN` — токен бота
   - `ADMIN_ID` — ваш telegram id (целое)
4. Запустите: `python main.py`

## Структура и заметки
- БД SQLite хранится в `/mnt/data/case_bot.db` по умолчанию (можно изменить `DB_PATH`)
- Функции и логика простые — для продакшена рекомендую использовать ORM, миграции, валидность входных данных и обработку ошибок.
- Чтобы добавить кейсы / предметы: измените DB через sqlite3 или добавьте обработчики в админ-панель.
